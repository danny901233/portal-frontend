🎯 ReceptionMate Widget - Embed Example

This page demonstrates how to embed the ReceptionMate AI Agent Configurator in your website using an iframe.

📋 Integration Instructions

Copy the code below to embed the ReceptionMate widget in your website. Replace localhost:3001 with your production URL.

<!-- ReceptionMate AI Agent Widget --> <div style="width: 100%; max-width: 1200px; margin: 0 auto;"> <iframe src="https://your-domain.com/widget" title="ReceptionMate AI Agent Configurator" style="width: 100%; height: 800px; border: none; border-radius: 12px;" allow="microphone" ></iframe> </div>

🔐 Security Considerations

• Set frame-ancestors in CSP headers to control which domains can embed your widget
• Implement authentication tokens for customer-specific configurations
• Use HTTPS in production for secure communication

// Example: Passing customer context via URL parameters const customerId = "cust_123456"; const garageId = "garage_abc"; const widgetUrl = `https://your-domain.com/widget?customer=${customerId}&garage=${garageId}`; // Embed with authentication <iframe src={widgetUrl} title="ReceptionMate AI Agent Configurator" ></iframe>
